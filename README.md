# 🚗 Car Marketplace App

A fully interactive car marketplace built using Vanilla JavaScript.

## 🔥 Features
- Dynamic car listing
- Click to view details
- Add to cart system
- Quantity management
- Remove items
- Clear cart
- Live total calculation
- Smooth UI animations

## 🛠️ Tech Used
- HTML
- Tailwind CSS
- JavaScript

## 🧠 What I Learned
- DOM manipulation
- State management
- Event handling
- Dynamic UI updates
